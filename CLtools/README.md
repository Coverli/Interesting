### 簡單的工具類插件

#### 插件功能

* 實現
    * 編碼
        * MD5、Sha1、Sha256、Sha512、AES、DES、RC4  (Crypto-JS加密庫)
        * 圖片base64編碼
        * 進制轉換
        * MarkDown編輯器
    * 查詢
        * HTTP狀態碼查詢
        * HTTP Content-type
        * TCP/UDP常見端口查詢
        * RGB顔色參考
        * ASCII對照表
    * 右鍵
        * 任何頁面通過右鍵菜單進入主頁
        * 選中文本后右鍵菜單在cdsn、bing、StackOverFlow進行快速搜尋
